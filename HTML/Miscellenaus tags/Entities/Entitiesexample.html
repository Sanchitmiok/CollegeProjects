HTML Entities

HTML entities are used to display reserved characters and symbols in HTML.

Common HTML Entities

Entity   used keyword(&) Character
& amp; (Ampersand)
< lt; (Less Than)
> gt; (Greater Than)
" quot; (Double Quote)
' apos; (Single Quote)
  nbsp; (non breaking space)

You can use these entities to display reserved characters in your HTML code. For example, < represents < and > represents >.
